HHG Python Interface documentation
==================================

Welcome to the documentation of the HHG Python Interface.  
This interface provides Python bindings to a C++ backend for simulating single-atom high-harmonic generation (HHG) in strong-field physics. It also includes plotting and convergence analysis tools.
